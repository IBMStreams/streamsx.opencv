OpenCV toolkit for IBM Streams

The OpenCV toolkit for IBM Streams allows applications to ingest and process images with the OpenCV (Open Computer Vision) library. This version of the toolkit is intended for use with IBM Streams release 4.1 and later.

The toolkit includes step-by-step instructions for installing it, the OpenCV library, and their dependencies, and for verifying that they are working correctly by executing some of its sample applications. It also contains documentation for the toolkit's operators. For details, please see:

  • Installing the Toolkit

  • Using the Toolkit

  • Sample Applications

  • Types and Operators

For information about the toolkit, please see http://ibmstreams.github.io/streamsx.opencv/.

For the toolkit's source code, please see https://github.com/IBMStreams/streamsx.opencv/.
